# Submission Package

This folder contains the written submission material and visual assets for Spark City Wallet.

## Files

- `submission-form.md`: form-ready project title, short description, structured answers, tags, and repository fields.
- `demo-video-script.md`: 60-second product/UI video script.
- `tech-video-script.md`: 60-second technical video script with code and diagram references.
- `diagrams/architecture.mmd`: high-level system architecture.
- `diagrams/context-to-redemption-flow.mmd`: customer journey from context sensing to QR redemption.
- `diagrams/privacy-ai-data-flow.mmd`: privacy-preserving AI and data flow.
- `diagrams/merchant-event-intelligence.mmd`: merchant dashboard and event-intelligence flow.

## Suggested Uploads

Upload the two MP4 videos separately. For additional visuals, export the Mermaid diagrams as PNG or PDF, or upload the `.mmd` files if the platform accepts source diagrams.

Before final submission, update the GitHub Repository URL in `submission-form.md`.
